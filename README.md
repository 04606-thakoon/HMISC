# 天主教新竹教區移民服務中心 網站專案

## 1. 基本資訊 (Basic Information)

- **客戶名稱**: 天主教新竹教區移民服務中心 (Hsinchu Diocese Migrants Service Center)
- **客戶負責人**: 永莉 (Jessi)
- **網站目標**:
  - 提供正確資訊，幫助移工。
  - 募集物資與善款。
- **實作方法**: Tailwind CSS + React
- **注意事項**:
  - **易用性**: 介面需容易操作，讓移工能輕鬆閱讀。
  - **吸引力**: 設計需具吸引力，增加使用者停留時間。
  - **RWD**: 必須支援響應式網頁設計 (Responsive Web Design)。

---

## 2. 網站架構 (Website Architecture)

### 1. 關於我們 (About Us)
- **主任的話**: 建立信任、傳達使命。
- **歷史沿革**: 組織大事紀。
- **服務據點**: 包含 Google Maps 導航功能。

### 2. 移工專區 (Migrant Worker Zone)
- **權益與法規**: 移工相關法規、權益須知。
- **常見問題 (FAQ)**: 針對移工在臺生活、勞動糾紛的解答。
- **緊急求助**: 聯絡方式、庇護資訊。

### 3. 雇主專區 (Employer Zone)
- **聘僱法規與注意事項**: 雇主法律責任、聘僱流程。
- **文化與溝通**: 雇主與移工相處建議。

### 4. 最新消息 (News)
- **活動報導**: 活動花絮與預告。
- **個案故事**: 具感染力，用於募款。
- **法規解析**: 政策變動的深入報導。

### 5. 支持我們 (Support Us)
- **金錢捐款**: 線上捐款功能、銀行帳號資訊。
- **物資捐贈**: 需有社工可更新的「急徵清單」。

### 6. 聯絡我們 (Contact Us)
- **聯絡方式**: 電話、Email、地址、社群連結。

### 7. 專案計畫 (Projects)
- **移工數位遠距學習**: 讓移工更了解網站的方向。
- **移工職涯發展**
- **文化交流活動**

### 8. 語言 (Languages)
- **分流多語系**: 網站需提供多國語言支援。

## 4. 核心技術與開發規範 (Core Technology & Development Standards)

### 技術堆疊 (Tech Stack)
- **語言**: TypeScript (Strict Mode)
- **前端框架**: React
- **樣式系統**: Tailwind CSS (PostCSS)
- **建構工具**: Vite
- **部署平台**: GitHub Pages

### 開發目標
- **高效能**: 優化靜態資源加載，確保快速的首次繪製 (FCP)。
- **可維護性**: 嚴格的 TypeScript 類型定義，模組化組件設計。
- **最佳實踐**: 符合 React Hooks 規範，使用最新的 React 18+ 特性。
